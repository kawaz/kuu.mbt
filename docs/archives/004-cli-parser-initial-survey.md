# CLI 引数パーサ初期調査

## 既存ライブラリ

### TheWaWaR/clap v0.2.6（最有力）

- 無限ネストサブコマンド
- グローバル引数
- 環境変数デフォルト
- カスタム Value trait

### Yoorkin/ArgParser v0.2.1

- 簡易版パーサ
- 基本的なオプション解析のみ

## ユーザーの好みとの適合度

| 要件 | clap v0.2.6 | ArgParser v0.2.1 |
|---|---|---|
| サブコマンド構成 | 対応（無限ネスト） | 非対応 |
| グローバルオプション | 対応 | 非対応 |
| 環境変数デフォルト | 対応 | 非対応 |
| `--help` セクション分け | 部分対応 | 非対応 |
| `--` セパレータ | 非対応 | 非対応 |
| `--no-xxx` 反転フラグ | 非対応 | 非対応 |
| completion 生成 | 非対応 | 非対応 |
| 引数なし → ヘルプ表示 | 非対応 | 非対応 |
| 環境変数ヘルプセクション | 非対応 | 非対応 |
| ロングオプション基本 | 対応 | 対応 |
| オプション位置自由 | 対応 | 部分対応 |

## 非対応の重要機能

- `--` セパレータ（メイン引数とオプションの分離）
- `--no-xxx` 形式の反転フラグ
- completion 生成（zsh / bash / fish）
- 引数なし実行時のヘルプ自動表示
- 環境変数の専用ヘルプセクション

## 方針

- **フルスクラッチを基本に検討中**
- 大規模な他言語パーサ調査を実施予定（別ドキュメントにまとめる）
- clap v0.2.6 をベースにフォーク＋拡張する案も検討対象
