# MoonBit 現状まとめ

## 基本情報

- **バージョン**: 0.8.0（2026年2月時点）
- **1.0リリース**: 2026年前半予定
- **ステータス**: ベータ（2025年6月〜）、構文安定、仕様変更はRFCプロセスを経由

## パフォーマンス

- **コンパイル速度**: Rustの10〜100倍高速

## バックエンド

| バックエンド | 成熟度 | 備考 |
|---|---|---|
| wasm-gc | 最成熟 | 主要ターゲット |
| JS | 安定 | ブラウザ/Node.js向け |
| ネイティブ（C+LLVM） | 実験的 | C言語出力→LLVMコンパイル |

- **moon**: Rust製ビルドシステム（AGPL ライセンス）

## 0.8.0 の主要変更

- **バックトレース**: エラー発生時のスタックトレース出力
- **declare キーワード**: 外部宣言の新構文
- **struct コンストラクタ**: 構造体の簡潔な初期化
- **Debug トレイト**: デバッグ用文字列表現
- **effect 推論廃止**: 明示的なエフェクト指定が必要に
- **moon.pkg DSL**: パッケージ設定の新しい記述方式

## mizchi の活動

### プロジェクト

- **luna.mbt**: Solid.js/Qwik インスパイアの宣言的 UI フレームワーク。SSR + ハイドレーションに対応
- **js.mbt**: JS FFI バインディング集。React バインディングにより SPA 動作を達成
- **markdown.mbt**: インクリメンタル Markdown コンパイラ。2万文字で60fps描画を実現
- **vite-plugin-moonbit**: Vite に MoonBit コンパイルを統合するプラグイン
- **moonbit-practice**: Claude Code 用スキル
