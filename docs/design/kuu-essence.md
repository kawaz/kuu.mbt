# kuu

投機実行と最適解釈で曖昧さを構造的に解決する CLI パーサエンジン。

## 特徴

- **投機実行+最適解釈**: 各コンビネータが消費可能な範囲を試し、最も妥当な解釈を採用する。定義順序に依存しない
- **簡単に使えて堅牢**: 普段使いはシュガーで気軽に。堅牢さは内部の直交コンビネータが保証
- **型安全**: 定義時の型がパース結果までそのまま保持。ダウンキャスト不要
- **直交コンビネータ**: alias / clone / link / adjust の合成で複雑なオプション関係を宣言的に表現
- **4層アーキテクチャ**: Core → bridge → KuuCore → DX API
- **多言語**: コアは MoonBit 製。各言語に最適な形式（WASM / embed / FFI）で提供し、DX 層で各言語のイディオムに馴染ませる
